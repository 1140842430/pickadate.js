The Themes§

There are two base themes available: default and classic. Check the readme file to see which files to use for each theme.

The themes are generated using Sass and can be customized by changing the variables set in the _raw/lib/themes/_variables.scss file. Take a look at the notes on compiling Sass using Grunt.

Default§

The default date picker:

<link rel="stylesheet" href="lib/themes/default.css">
<link rel="stylesheet" href="lib/themes/default.date.css">

The default time picker:

<link rel="stylesheet" href="lib/themes/default.css">
<link rel="stylesheet" href="lib/themes/default.time.css">

Scroll into view§

For the classic theme, the API is used to create this snippet to scroll the picker into view:

picker.on('open', function() {
    scrollPageTo(this.$node)
})

function scrollPageTo($node) {
    $('html, body').animate({
        scrollTop: ~~$node.offset().top - 140
    }, 150)
}

The scrollPageTo function can be elaborated further to only scroll the page when the picker is out of view.
