browser support§

Supports all modern browsers – and IE7+.

Make sure to also include the legacy.js file along for the oldie browsers.

Specifically, v3 has been tested with Chrome MWA, Firefox MWA, Opera MWA,
Safari Mi, Internet Explorer W, Blackberry 6.0+ and Android 3.2+ browsers.

M = Mac, W = Windows, A = Android, i = iOS
