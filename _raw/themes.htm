The Themes§

There are two base themes available: default and classic. Check the readme file to read up on files for each theme.

Default§

The default date picker:

<link rel="stylesheet" href="{%= dirs.dest.themes %}/default.css">
<link rel="stylesheet" href="{%= dirs.dest.themes %}/default.date.css">

The default time picker:

<link rel="stylesheet" href="{%= dirs.dest.themes %}/default.css">
<link rel="stylesheet" href="{%= dirs.dest.themes %}/default.time.css">

<link rel="stylesheet" href="{%= dirs.dest.themes %}/classic.css">
<link rel="stylesheet" href="{%= dirs.dest.themes %}/classic.date.css">

The classic time picker:

<link rel="stylesheet" href="{%= dirs.dest.themes %}/classic.css">
<link rel="stylesheet" href="{%= dirs.dest.themes %}/classic.time.css">
