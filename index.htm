And customize it as you please…

the time picker§

Initiate it $('.timepicker').pickatime() then pick a time:

And customize it as you please…
Time intervals, time formats, and lots more.

features§

Works with jQuery 1.7+, BEM style class naming, Sass based styling

browser support§

Supports all modern browsers – and IE7+.

Make sure to also include the *.legacy.js file along for the oldie browsers.

Specifically, v3.0 has been tested with Chrome (MWA), Firefox (MWA),
Opera (MWA), Safari (Mi), Blackberry 6.0+ and Android 3.2+ browsers.

M = Mac, W = Windows, A = Android, i = iOS
